# Magical Wooden Clock Scene

An interactive Flutter application that brings a cinematic Pixar-style story to life. Set in a serene Indian village courtyard bathed in soft morning sunlight, the app features a beautifully rendered wooden clock. When touched, the clock instantly stops, and magical blue particles burst forth, creating an emotional and visually stunning experience.

## Features
*   **Interactive Wooden Clock**: An animated analog clock that stops when tapped.
*   **Magical Blue Particles**: A custom particle system that emits glowing blue particles upon interaction.
*   **Cinematic Atmosphere**: Styled with soft morning lighting, gradients, and immersive visual cues.
*   **Responsive Layout**: Adapts to mobile, tablet, desktop, and web.

## Tech Stack
*   Flutter (Dart)

## Setup
1. Install Flutter.
2. Clone this repository.
3. Run `flutter run` to launch the application on your preferred device.
